# Tracktivity Notifications

Public notification feed for Tracktivity dApp.

## Structure
- `notifications.json` - Notification data fetched by the app
- Updates deploy automatically via Vercel

## Notification Types
- `update` - App updates
- `promotion` - Paid promotions
- `maintenance` - System maintenance
- `polymarket_starred` - Starred user activities (future)
- `custom` - Other announcements
